怎样连接电脑下载图片

发表于 2006-09-28 | 分类于 所有文章

1 建议使用三星的原装数据线，数据线型号：PCB113UBCC 2 在手机与计算机连接之前,需要安装应用程序（Samsung PC Studio PIM & File Manager 2.0)和驱动程序（USB Driver），它们都可以在随机光盘中找到,或登陆” www.samsung.com.cn"产品--下载中心，下载。（注：在安装驱动程序之前，手机与计算机不要连接） 3 软件都安装成功后，手机开机使用数据线连接计算机,系统会发现新的硬件,并弹出新硬件安装向导，按照向导提示点击”自动安装软件”，系统就会自动查找驱动并安装它，过程中系统会提示”此软件没有通过Windows的徽标测试”点击仍然继续安装,然后完成。系统会提示此硬件安装完成并可以正常使用了。（注：若完成后仍然发现新硬件并弹出新硬件安装向导的话，则需要按照上面步骤重复操作直至完成） 4 打开电脑的”控制面板”中的”电话和调制解调器选项”然后在”调制解调器”选项中,会看到”Samsung Mobile USB Modem”以及连接所使用的端口号。 5 打开”PIM & File Manager 2.0”，点击菜单”文件””设置”。速度选115200,端口要与调制解调器中的一致，然后确定。再次点击”文件””重新载入”–”全部重新载入”,此时软件会读出手机中的电话簿短信息等资料，软件的下面也已显示了手机型号SGH-D508,表明手机已经与计算机连接成功。 6 打开菜单”工具”–”File Manager”,在出现的对话框中，点击”＋”打开”我的电话”选中”Images”下面的”download Images”文件夹，然后在本地存储中，选择图片所在的文件夹,在界面的右下角选中所要传输的图片文件，点击蓝色向上的箭头,等待传输过结束后图片就储存到手机中了。
