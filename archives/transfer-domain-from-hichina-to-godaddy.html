域名从万网转到GoDaddy经验谈

发表于 2012-03-14 | 分类于 网络爬虫

国内的站长一般都对国内的IDC们深恶痛绝，他们一方面是犬牙，另一方面是无良的商人，为了在国内生存无所不用其极。 2008年春，我在万网某二级代理商处注册了国际顶级域名songruihua.com，当时刚刚接触个人建站，对国内形势不够了解，不慎上了贼船。域名续费费用从55元每年涨到60元，又涨到65元，并且国内的风声越来越紧张，停止解析、禁止转出的事例层出不穷，于是决定将域名转到国外。 本来希望转到name.com名下的，可发现需要PayPal或者信用卡，搞了半天不能付款，最后决定转到godaddy.com，因为接受支付宝付款。域名注册商的转移流程是这样的： 1.万网将域名转移密码提供给我 2.交费后，我将密码告知GoDaddy 3.GoDaddy凭密码去万网提出接收域名注册业务 4.万网同意放行 万网由于信誉不佳，客户不断流失到国外，为了保持高利润，他们会想方设法地阻挠用户转出域名。第一步的转移密码，用户就很难获得。在整个域名注册行业中，存在四个主体，分别是ICANN、万网、二级代理、域名持有者。.com、.net、.org等国际顶级域名的注册业务由ICANN直接管理，ICANN接受万网、Godaddy、Name等域名注册代理商的注册，同时也制约这些代理商。万网主要在国内开展中小企业和个人的域名注册业务，大企业的域名多交由国外代理商注册，万网为了拓展业务，在国内发展了许多二级代理，并制约这些二级代理。用户如果想在国内注册域名，既可以直接找万网，也可以找这些二级代理，但是直接在万网注册费用要比二级代理贵一倍，所以国内个人注册域名多在二级代理处注册，但是在ICANN看来，仍然是在万网名下。 首先，需要明确一个基本事实，域名持有者所注册的域名本质上是在ICANN注册的，谁拥有某个域名均以ICANN的数据为准，万网及其二级代理只是在ICANN和域名持有者直接充当桥梁的作用，域名持有者如果不满意某个代理商，完全有权利随意更换中意的代理商，如果代理商出于利益考虑阻挠域名持有者更换代理商，ICANN有权利和义务介入。 万网在其官方网站中声称的具体办理步骤如下：

（1）请您登陆万网网站首页－客户中心-资料中心，打印《万网国际域名转移注册机构申请表》（国际英文域名、新顶级域名、国别域名域名转出均使用此表格查询密码）； （2）将填写完整的申请表打印并加盖域名所有人和中国万网会员ID所有人公章，连同域名所有人营业执照副本复印件一同邮寄到万网；域名所有人为个人或中国万网会员编号所有权为个人的，请附上个人身份证复印件(如是第二代身份证请提供正反两面)； （3）如果相关资料合格，我们将会把转移密码发送到域名所有人的邮箱中；我们收到您邮寄的材料后会配合转入方做好转移确认工作，同时请您尽快与转入方联系办理转移业务。 我公司的邮寄地址是： 北京：东城区鼓楼外大街27号万网大厦3层（业务处理部 收） 邮编：100120。 注：如果域名所有人中文、英文为您公司（个人）的英文简称或者无效信息，请您提供购买域名时万网开的发票复印件，我们需要核实您的发票抬头。具体请您致电客服中心 如资料经审核后，存在下列现象将不予转出，相关责任由申请单位（或个人）单独承担： ① 该域名的当前服务商不是万网； ② 公章（身份证）与该域名的所有人登记信息不符（以WHOIS中的记录为准）； ③ 公章（身份证复印件）不清楚或没有加盖公章（附上身份证复印件）的； ④ 项目填写不完整，或域名信息与WHOIS信息不符； ⑤ 该域名未交纳域名注册费（欠费）。

除了上述条款，万网还曾经要求先续费再转出，估计是被投诉了，这条被撤下了。 可见，万网为了阻挠域名持有者转出域名，设置了很多障碍：打印申请表、盖公章（复印身份证）、复印发票、快递文件、信息不符、复印件不清楚、信息不完整、欠费。 用户只有完全按照万网的苛刻要求办才行，而且这中间万网可以非常容易地拖延时间，比如明明收到快递，却声称没有收到，明明文件清楚，却声称字迹不清，各种理由，各种无耻。其实，只需要确认一点，域名持有者只要能证明自己就是WHOIS信息中的注册人既可，所以把身份证扫描件用WHOIS信息中的邮箱发给万网就行，其他的一切无理要求都不用管他。 万网作为ICANN管辖下的企业，其行为受ICANN制约，对付万网最有效的手段还是去ICANN投诉他们，投诉地址： http://reports.internic.net/cgi/registrars/problem-report.cgi 附带一下表格的填写方法： 1、Your name 栏目填写您的名字 2、Your organization(if any)栏目填写组织机构，为可选填写 3、Your email address 栏目写上您的email地址，方便regestrar跟您联系 4、Your phone number(optional)栏目可选填写您的电话 5、Specific domain name 栏目写上您要投诉的域名，这里只能填写一个，因为提交时候会自动从数据库查询该域名的注册商regestrar. 6、Please select your issue 写上您将要投诉的问题类型，我们选Transfer Problems - Auth Codes Locked Domain Fraudulent Transfer Registrar Denied Transfer，表示是域名转移相关的问题。 7、下面的brief summary写上您的投诉事件简要描述即可。最好把问题写清楚简要。然后提交即可。 综上，正确的操作步骤是： 1.找二级代理，并将身份证扫描件（加水印）电邮给他们 2.找万网，并将身份证扫描件（加水印）电邮给他们 3.去ICANN投诉万网，对，刚联系完他们就去投诉，不要对他们抱有幻想 4.等着万网的转移密码 5.将转移密码告诉GoDaddy 6.等着万网同意放行，最长5天之后如果还不放，打电话骂他们 万网3月8号给我发邮件说，到3月13号下午15：49分，如果我没有取消域名转移，就放行，结果13号晚上还没有动静，当晚我就打电话给万网把话务员小弟骂了一顿，14号上午万网就放行了。以下是 Wed, 14 Mar 2012 05:47:22 UTC时刻的WHOIS信息。

Domain Name: SONGRUIHUA.COM Registrar: GODADDY.COM, LLC Whois Server: whois.godaddy.com Referral URL: http://registrar.godaddy.com Name Server: F1G1NS1.DNSPOD.NET Name Server: F1G1NS2.DNSPOD.NET Status: clientDeleteProhibited Status: clientRenewProhibited Status: clientTransferProhibited Status: clientUpdateProhibited Updated Date: 13-mar-2012 Creation Date: 03-apr-2008 Expiration Date: 03-apr-2014

PS：域名转移到国外是出于政策原因，而非技术原因。所以DNS服务器不要转移到国外（如果网站的访问来源主要来自国内的话），使用GoDaddy的DNS服务不仅国内解析速度慢，还可能被墙，推荐使用国内的DNSPOD解析域名。
