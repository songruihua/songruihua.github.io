囧：Win8开着任务管理器就无法安装腾讯TM

发表于 2011-09-14 | 分类于 小品软件 ， 扯淡龙门

怪事年年有，今天特别多！除了在Win8下迅雷导致IE10浏览器崩溃的问题、QQ2011无法群聊的问题之外，Win8之家还和大家分享了另一个囧事，那就是：在Windows 8开发者预览版里如果您运行着任务管理器，那么，就无法正常安装QQ2011。 想知道怎么回事么？ 看看下面的截图，您就明白了——

在新的Windows8操作系统里面，Windows Task Manager 就是任务管理器的名字从Vista、Win7时代的taskmgr.exe 更名为TM.exe了，而这个TM.exe和我国腾讯公司QQ的姊妹软件TM重名。 腾讯QQ和TM同属一个软件核心，在安装过程中检测的方式太简单，只看进程里面是否有相同的名称，这个过失对腾讯这样的大公司来说是不应该发生的，属于开发者的粗枝大叶导致的，哪怕用Windows 自带的API来判断也比这个要好很多。相信QQ的后期版本肯定会改进这个问题，毕竟，你不能指望人家微软的Windows里面的TM.exe去改名。其实都不用改名， 就是判断机制QQ要加强一下。 好了，找到了问题的原因就简单了，大家记得，在Win8里面安装QQ或者TM的时候，要把任务管理器先关掉……（文/Win8之家） 这根本是腾讯脑残，我们不得做一个艰难的决定，我们不得做一个艰难的决定，腾讯把进程名改成TMD 就OK了。腾讯的开发者从来不会使用windows的API，qq旋风的桌面快捷方式都建不好，如果移动了桌面，它非要自己造个桌面文件夹，把那个该死的快捷方式放进去！根本不用协商，一定是腾讯改，腾迅V5。
