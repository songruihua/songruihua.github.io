我的壁纸——For Windows 2003

发表于 2008-08-16 | 分类于 小品软件

上面的壁纸是在微软官方Windows Server 2008网站上下载的壁纸，形象是08的机器人，我添加了自己的logo、我的域名、firefox的logo（像一个篮球）、我用的杀毒软件ESET Smart Security的logo，又把右上角的Microsoft标志灭了，又把Windows Server 2008的“8”去掉了一半，成了2003，大功告成！ 最近刚刚换上Windows 2003企业版，确实比较好用。体会最深的一点是在我的512内存笔记本上，打开Firefox上会网，打开Photoshop做会子图，再打开2个TM，让这些软件共同工作一会，然后都关了。猜怎么着，系统依然十分流畅，在桌面打开属性菜单，瞬间完成，再看内存占用——46%！我刚开机时可是55%，开了这么多软件内存竟然释放的这么快，同样的操作在XP上进行的话，只有重启了，非如此系统不能流畅运行（又仨小时后也能释放内存，但我等不了）。 Windows2003拥有比WINXP更加强大的内存管理，在稳定性、程序响应速度方面更超越XP。是因为Windows2003 的设计初衷导致个人用户需要经过很多手段、方法调整后才能正常使用这个如此强大的系统，如果不这样，从2000，XP转向2003的用户会感觉很不习惯，致使很多个人用户对Windows2003望而却步。 现在，你有了新的选择了！ Windows2003个人精简版将令你完全改观！它的易用性、定性、安全性会让你感到兴奋！在众多的精简版本中，又以盛世雄风论坛的冷风精简的03企业版为最成熟的版本，冷风多年接触2003的精简工作，其作品十分适合已经熟悉XP操作的用户使用，其最新作品是080808版。
