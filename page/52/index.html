WINDOWS VISTA AERO VS LINUX UBUNTU BERYL

发表于 2007-04-18 | 分类于 小品软件

　　上面的视频是最早出现在YouTube上，后被转载于Tudou上的，因为后者在国内的速度快于前者，故链接到后者。次视频的制作者对比了Windows最新版本vista中的3D特效aero和Linux诸多版本之一的UBUNTU中的特效BERYL。对比一下。最后作者搬出了微软公司的Windows98发布会会上的Windows系统出现蓝屏的戏剧性的一幕，特别强调了盖茨身边的微软员工说的“WOW”,vista在选宣传时也说“WOW”， vista也会蓝屏吗？问问盖茨。

　　如果你不是精通电脑知识，那么或许从你第一次接触电脑，你在屏幕上看到就是由一家叫做微软的公司的产品——鼠标、窗口、左下角的开始按钮、右下角的时间显示，还有经常莫名其妙自己关闭的IE浏览器和那个著名的蓝屏。

　　或许你大学所学习的一切都是基于微软强加给我们的产品——windous基本操作、word、excel等等，你不得不在微软为我们设置好的成千上万的小按钮中苦苦寻找自己想要的那个，你不得不为通过国家计算机等级考试而在机房中苦苦校习一整天的frontpage，因为似乎考试、网吧、电脑的一切都是微软的，甚至我想在一些人的头脑中一想到电脑，首先出现的不是主机、电路板、屏幕、鼠标，而是那个统治全球超过95%电脑操作系统的windows桌面，以为这就是电脑。

　　不，绝对不是这样的。你或许不了解，如果你在中国大陆购买一台组装好的品牌机，按照中国法律，这些电脑是必须预装正版操作系统的，而几乎100%的组装品牌机预装的是Windows系统，例如如果你现在购买预装Windows Vista 家庭高级版的品牌机，那么你的购机款中将有好几大百是支付给微软以换取正版Vista系统的，可以说如果中国大陆地区的电脑装正版Windows系统的比例增加一倍，那么中国电脑现代化进程将会延后很多年，因为微软的产品是私人软件——未经授权的获得或编辑都是非法的。

　　而linux则不是，他是开源的，任何有能力的人都可以在其基础上进行编辑、修改，或者出售自己的成果，如Novell,RedHat,当然你也可以免费送给别人。

　　Ubuntu就是如此。Ubuntu是很多linux系统桌面中的一种，它以类似于Windows的用户界面沟通了用户和后台的Linux，让LInux更加简单易用。linux是免费的，在其基础上的Ubuntu也是免费的，甚至是主动送到你手中的赠品。根据Nelson Mandela的解释，Ubuntu有诸多含义：尊敬、乐于助人、分享、社区、关怀、信任、无私。 而其不同于windows的主要有以下几点：免费、简单、稳定、安全。

　　你只要有一定的英文水平，登录Ubuntu的网站，https://shipit.ubuntu.com/ ，说明自己的姓名和地址，几个月后几张Ubuntu光盘就会从国外的Ubuntu机构通过邮政寄送到你的手中，装入电脑，点击几次“下一步”，Ubuntu自动安装，一切都是免费的，邮资全面，你所要支付的就是给供电局的电费和电信商的上网费，其他Ubuntu全包，而Windows……正版，穷人别想；盗版，人民币十块！

　　windows之所以能推广，就是因为它把所有的操作步骤都分成了一个个按钮，让用户不断得完成琐碎的按钮操作，是，每一部都很简单，但不觉得这太繁琐吗?Ubuntu给你不一样的操作体验，但你可能会不习惯，使用时请不要试图让Ubuntu跟Windows学习，他们的确不一样，你所要作的就是抛弃一些微软灌输给你的东西，重新学习和适应一款新的操作系统，慢慢得就会体会到Ubuntu的好处。

　　至于稳定性。你是否经常在Windows下使用Ctrl+Alt+Del来关闭“死”了的程序，看看这里http://youtube.com/watch?v=q7DhOt_5NXA，想想如果是在Windows里，你需要按几次Ctrl+Alt+Del，看到最后相信你会惊呼Ubuntu的稳定性，其实大多数基于linux的操作系统都很稳定。再想想视频最后的微软员工为什么当着大老板盖茨的面说“WOW”，他已经不再微软工作了，不能怪他，不是吗？

　　安全。针对Windows的病毒数是：问问盖茨。针对Ubuntu的病毒数是：约300。加装卡巴斯基后约为0。

　　再等几个月我的Ubuntu光盘就将从荷兰飞到我的手中。

　　我昨天帮一个比我更菜鸟的小子chopman从google的blogger把一个模板搬迁到blogbus上了。虽然费了我半天的劲，但路遇知音，帮忙是应该的。

　　我也曾经深爱着Blogger，把它视为一生的伙伴，但是我无法容忍|防|火|长|城|的无耻行径，所以我不得不放弃了Blogger来到blogbus上安家， chopman有着和我类似的遭遇，也是被逼到bus上的，因为bus是国内模仿blogger最像的一家bsp，这一点可以从bus的自定义模板看出——把bus的标记中的“entry”改成“post-”几乎就成了活脱脱一推blogger标记。

　　我离开了blogger却固执地坚持使用blogger的模板，虽然要抛弃熟悉的blogger标记重新学习bus的模板设计规则，但是我没有怕麻烦，一定得用上blogger的模板，争取除了把spot换成bus其他的什么也不要变，还把原来的http://songruihua.blogspot.com设置了自动跳转到新博客，chopman也恋恋不舍blogger的harbor模板，于是我帮他整过来了。

　　帮了一个同病相连的中国人。
