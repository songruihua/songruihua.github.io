工商银行U盾全攻略 (兼容Windows 7 多图)

2009-04-03 | 小品软件

本文的目标受众是初次接触网络银行的所谓“小白”，请自诩高手的“老鸟”自行绕过。

1.什么是U盾？ U盾是中国工商银行推出的网上身份认证工具。工行会给每个申请U盾的用户一个数字证书，就跟身份证类似，当有钱要从账户中转出时，工行会要求用户出示该数字证书以证明其合法身份，但数字证书不能像身份证那样拿在手里，于是工行推出了一个类似优盘的移动存储设备用于存放数字证书，这个“U盘”就是所谓的U盾。 2.如何拿到U盾？ 条件一：身份证 办U盾需要到柜台，没身份证不行。 条件二：一个工行账户 已经有工行账户的话可以不新开户，但鉴于即使是U盾也有被盗用的可能，所以建议为网上交易开一个专门的账户，平时不要存太多的钱，需要交易时临时存入，立马支付，存有几十万现金的账户最好不要在网上留下任何痕迹（压根就别再键盘上输入），要取钱去银行柜台办理，这样保险。 服务台的人会给你两张表，一张开户申请表，一张网上银行申请表，在后一张表上不要勾选“电子口令卡”，要勾选“U盾”。等叫到号了去柜台交上两张表，说明开户办U盾，然后等着签N次字，输入N次密码。 3.如何为网上支付做好准备？ 全新的U盾里面是没有数字证书的空盘，你的数字证书还在工行的网络系统中，你需要将此数字证书导入到U盾后，U盾才能成为一个身份认证工具，由于事关财产安全，工行将这一过程弄的相对复杂，所以不要妄想回家后插上U盾就能支付这样的美事了，一步一步来吧。我对电脑算是比较熟悉的了，第一次用也被搞到头大，网上的解释都不是很直观，所以我决定配图详细介绍一下导入过程，以方便后来人。 第一步：安装软件 你不必要知道或理解这连个软件是干什么用的，下载并安装吧。 登录 软件一：工行目前提供5种证书驱动程序，请根据你领到的U盾的品牌和型号（不知道的话看说明书）下载相应程序。我持有的是品牌是捷德，其他品牌如出现使用障碍请致电中国工商银行或到柜台咨询。 软件二：控件程序是必须要下载的，并且U盾只能在IE下使用，如果你不知IE为何物那么你肯定不会为此烦恼的，放心吧。 以管理员帐户登录Windows，关闭所有窗口，逐一安装这两个程序，目前我的亲身经历是Windows Server 2008 32位 标准版、Windows Vista 32位 家庭基础版，Windows 7 32位 Build 7068，Windows 7 64位四个操作系统可以正常安装，我已经不用XP所以没有试用，应该是可以用的。 第二步：登录https://mybank.icbc.com.cn/icbc/perbank/index.jsp

请记住工行的官方域名是 icbc.com.cn，以后如果遇到 lcbc.com.cn，iobc.com.cn 这样的欺诈网站千万不要输入自己的账户密码，这些网站的网页布局跟工行官方网站一模一样，能蒙倒不少人，所以大家进行网上金融操作时一定要格外注意网站的域名。

第二步：输入卡号和密码登录

关于密码 工行的卡和网上银行（带U盾）共涉及4个不同的密码

  1. 取款密码：仅在银行柜台或ATM机上使用，网上银行业务不能用；
  2. 登陆密码：网上银行系统登陆
  3. 支付密码：网上银行存款的转出（含支付、转账、缴费等业务）；
  4. U盾密码：插入U盾后，用于确认U盾合法使用的密码。

第一个密码只能在网下使用，后三个只能在网上使用，互不干扰。 U盾支付要求用户满足四个条件：账户号码、登录密码、支付密码、U盾实体。知道账户（卡号）和登录密码可以登录工行的个人网上银行后台进行余额查询等操作，但是如果想让钱从账户转出必须再提供支付密码和U盾实体，否则不能转出，这也是为什么U盾的安全系数如此之高了，网上什么黑客、木马、病毒再牛不也就是偷偷你的账户密码等“数字”吗，只要U盾仍在你手中盗贼就就奈何不了你的人民币。

第三步：修改密码 第一次登录管理后台你需要修改登录密码，支付密码暂时不需要改，最好不要将这两个密码设为一样的，支付密码必须是6～8位，但登录密码可以很长（比如我的就是十多位密码） 第四步：选择菜单中的“U盾管理”–“U盾自助下载”，根据提示插入U盾，输入在用户柜台设置的支付密码，最后提示导入成功即可。 4.如何使用U盾支付？ 在不同的地方消费，支付方法略有不同，但大体可归纳为： 1.选择银行（当然是工行了） 2.输入账户（卡号） 3.输入转账金额 4.跳转到工行网上银行（此时一定要注意域名是否是icbc.com.cn） 5.输入登录密码（可能没有这一步） 6.要求插入U盾 7.要求输入支付密码 8.提示支付成功 9.立刻将U盾拔出（无需像U盘那样“安全移除硬件”，直接拔掉）

没带钥匙，干脆去大活看电影，平安夜应该放关于圣诞的电影，令人高兴的是今晚电影票买一送一——一张票看两场。 不知从何时大陆开始流行这种西方节日，不过我深切地感受到这种气氛是在邯郸上学时，来天津后氛围更浓烈了，大城市加上身处青年人聚集的大学校园，一直不想参加西方神灵的生日派对的，今天看电影时倒是深切的感受了一下。 我觉得，其实西方现在过圣诞也不仅仅是因为信仰，而是像中国春节一样，每年固定一个节日享受家庭生活的快乐，感受亲情。不过，在大陆已异化为青年人聚会的理由之一。

流氓网络电视PPS

2008-12-20 | 小品软件

当两台电脑都开着网络电视的时候，我的电脑就会在网上蜗行了。不得已也装了网络电视，先后下载了PPLive和PPStream两款网络电视。 记得以前也装过此类软件，觉得广告没那么多，这次装后，发现在选中节目后会先放一段广告，时间大约有30秒，要说不加广告企业无法生存，这也无可厚非，但是企业以牺牲用户体验为代价就得不偿失了，新浪、搜狐等门户以前在首页上弄弹窗广告、浮动广告，现在更是发明了什么背投广告、悬浮广告，严重干扰访问者视线、腾讯强制弹出新闻，并且不允许非会员用户关闭这个东西，明知会干扰用户，还要藉此来向用户要钱。印象中，优酷的广告不错，点开一个视频，只会在用户暂停的时候和视频旁边显示广告，不会在视频播放之前出广告。 互联网企业的这种行为，导致网上出现了很多“去广告版”和补丁，搜索谁家的软件去广告版最多，说明谁家的软件最不顾用户体验，现在我更是发现了一个我认为最“流氓”的广告模式，来自PPS。 无论什么广告，都是在用户访问该网页或打开某软件后才会出来，但唯独有一家企业——PPS的老板，把个PPS搞得无比恶心，只要是安装了PPS，即使不开着看片，也会蹦出广告，原来以为是PPS强加的一个广告进程，后来发现原来是一个屏幕保护程序，在安装时默认是安装的，同时默认的还有随windows自动启动。真是无所不用其极。唯一不及腾讯流氓的是，腾讯把QQ放在了开始菜单里Windows Update旁边，PPS没有。

极速文件搜索——Everything

2008-12-17 | 小品软件

当电脑用久了，里面的东西就多的找不到了。经常是只知道在哪个分区，然后就在分区根目录搜索，而 我有45G的分区已近快要满了，搜索会持续很长一段时间。相信很多人经常使用XP的搜索功能，也会感觉速度太慢。Vista的搜索已经很不错了，不过只适合电脑配置比较好的人用，如果配置一般，关闭Vista的搜索索引能提升不少性能。 于是，我就试着找了找相关的软件。经验告诉我，微软的产品使用的人是最多的，但通常不是最好用的，当然对于商用计算，微软可以找出稳定、兼容等一大推的借口，但对于个人来说，Windows的很多部件都有很多更好的替代者。 好了，我找到了不少的这类软件，大致分为两类： 一类是以Windows Desktop Search和Google Desktop Search为代表的“大型”软件，它们不仅能索引文件（夹）名，还能索引TXT,DOC等文件的内容，也能搜索电子邮件的内容（当然是下载过来的本地邮件了），代价就是索引会很大，我80G的硬盘，Google竟然创建了2G的索引文件，而且从安装到索引结束，持续了一两个小时，并且我发现，就算是把索引文件转移到系统盘之外，在还原系统后依然要重新索引。此类软件的速度一般，但比系统自带的速度要快不少。如果是经常进行文字工作的人，安装这个很不错，因为可以搜索到文件的内容。 另一类是以AVA Find，Locate32和Everything为代表的轻量级软件，通常也就1~2M的安装文件，只能搜索文件名，优点是索引快，80G硬盘索引过程都不超过5分钟，生成的索引文件不超过10M，搜素速度可以说只能以毫秒计算，很遗憾的是我没有那个精度的计时工具。在试用了以上三个软件后，我选中了Everything。 Everything目前支持Windows 2000, XP, 2003和Vista，并且

要求硬盘分区格式是NTFS，不支持FAT，FAT32格式

，查看分区格式方法是，打开“我的电脑”，右键单击某个分区，比如C:，点“属性”。 文件格式如果是NTFS就行，如果不是NTFS，建议找比较熟悉电脑的朋友帮忙弄，千万不要自己瞎弄，小心丢硬盘资料。 官方目前还没有推出中文版，不过这种搜索软件也不用怎么设置，英文过四级的应该都看得懂，现在官方网站（https://www.voidtools.com/download.php）上的稳定版本1.1不支持中文，测试版1.2支持中文，这里的不支持不仅是指用户界面不是中文的，而是不能索引名称中包含中文的文件（夹），

国内用户一定要下载1.2beta或以上版本，

，坚决不做小白鼠的请绕过。1.2安装文件仅300K左右，很小~ 1.安装过程很普通，一通下一步，就搞定，为了和系统结合得较紧密，也为了备份系统的方便，建议安装到系统盘，索引文件很小不用担心占用系统空间，而且索引的生成是实时自动且后台的，不用担心还原系统后索引失效的问题，80G硬盘生成索引用时不超过30秒，因为Everything用了和别的同类软件不同的索引技术，有兴趣的请自行搜索。 2.安装完成后，需要进行一下简单的设置，打开Tools>>Options. 3.General选项卡下，设置快捷键，即呼出搜索窗口的快捷键A+B，上面的列表选择A，下面的选B，我选择的是Win+S. 4.NTFS Volumes里可以设置索引哪些分区，一般来说资料不会放在系统分区（通常是C:），可以不索引它，一来可以减小索引文件大小，提高搜索速度，二来能减少搜素结果的干扰项。而其它分区最好全部索引。另外还能设置索引文件的路径，由于索引很快，文件很小，一般不用改这一项，除非有大量的文件需要索引。 选择需要索引的分区，把下面对应的四个方块全勾上。 5.Exclude list选项卡下可以设置排除路径，比如私人日记和我们邻国的某些短片。千万可不能让人给搜出来啊……不过小心搞成此地无银三百两啊！ 6.就这些了，下面秀一下： 我80G硬盘除系统分区外共有约11万个文件，天啊。 搜索hot的结果。 7.Everything支持正则表达式，可以用?和*，还支持“和”，“或”搜索，加上空格，即“A B”就能限制结果中必须同时包含A和B，加上“|”，即“A|B”就能实现“或”搜索，即A和B不管包括谁都会被搜索出来，更多的正则表达式请移步这里。还有更多高级功能，不过一般用户通常用不上，有兴趣的请移步这里。 8.资源占用。CPU占用只有在第一次索引时较大，平常使用时几乎可以忽略，内存占用就比较大了，一般在10~20M之间，我的本内存1.5G安装前后没感觉变慢，如果内存只有512M就不要用这个了，留着内存干别的吧，毕竟搜索又不是什么重要的事。

听张树仁教授讲桥梁，及其它

2008-12-13 | 又土又木

下午听了张树仁教授的讲座，收获没有。不是老先生讲的不好，而是我辈水平不够啊，张教授讲的是在南京什么学术会议上讲的东西，研究生都不懂，我怎么能懂，不过我倒是明白了，这桥除了要造，还要修，中国公路上的超载车辆，施工单位里的黑心包工头，都是我们桥梁专业学生的衣食父母啊，国家让一部分人去铺路造桥，还要派一部分人去补路修桥！ 张教授早年毕业于哈尔滨建筑工程学院并留校任教，曾任道路与交通工程主任，交通科学与工程学院院长，原哈尔滨建筑大学副校长。现任哈尔滨工业大学桥梁研究所所长，兼任黑龙江省公路学会副理事长，中国土木工程学会市政工程分会理事、城市桥梁学术委员会委员。 张教授可谓中国桥梁界数一数二的人物了，不知学院是怎么把他请来的，老先生从下午两点讲到四点多，依然很带劲，没有一点倦意，他今年都73岁高龄了，看来干施工的人身体就是好啊，我比较胖，有时会自嘲地说学土木就是为了在工地上减肥，要说也是，自打小学就让人给起跟某著名动物相关的外号，我还真的没瘦下来过，高考那阵也瘦了，不过肉都是从脸上掉的，家人一看顿生怜爱之心，可肚子上的肉就是不见少。 自打入校以来，就没见过什么知名专家来我们学校，也可能是我没看到海报，反正此次是我第一次听大师级的人物讲座，我怎么又表露出对学校的不满了呢，有时跟同学聊天时，有人提到，大部分大学生都对自己的学校不满，这个不满不是因为学校做了什么让学生不满的事，而是一种心态，上专科的羡慕本科大学，上二本的羡慕大本，上河工大的羡慕天大，上天大的羡慕清华，上清华的羡慕国外大学，不知道这是80后特有的还是古往今来学生的通病。 很喜欢一句话“生活得当段子过”(此处的段子指相声段子)，想想太阳在天上照着地球，地上的植物在生长，动物在你吃我来，我吃它，人也在人自己制定的规则下生活着，无论以怎样的心态都要在这世上活着，干嘛不高兴点呢。想想文革的时候被批判的那些人，有人因为心里上不能接受当众被批斗而自杀，有人痛苦不堪，也有人以乐观的心态看自己的遭遇，最后这种人真是让人敬佩啊，又想到了《贫嘴张大民》中结尾的时候，大民对他儿子说的一句话，大意是“什么时候别人把刀子放你脖子上非让你死你再死，否则你就快快乐乐地活着”，说到活着，又想起余华的《活着》，除了对战争的批判，我能看到的唯一一点有一定深度的东西就是作者希望大伙都快快乐乐地活着。 这世上的粥总是少的，和尚总是多的，还有人想喝两碗的，尤其是生活在当今这个社会转型期，能把生活当段子那样好好活着过的人越来越少了，当下活着尽管很累，我也希望累并快乐着。 我在标题后面加了“及其它”仨字，因为我跑题了。
