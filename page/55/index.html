学期总结

发表于 2007-01-26
| 字数统计: 406

打算明天一早六点半坐上649，要回家了。

想想去年9月爸妈把我从家送到天津仿佛才过去不久，或许是大学的第一学期过的太空的缘故吧，总较着时间过得太快，感觉大学的第一学期怎么这么没意义啊，一点值得怀念和自豪的事都没留下。临走了，想简单总结一下这段日子的得失。

　得
我初尝了传说中的大学生活，砸么出了那么一点滋味儿，似乎大学就是混文凭的地方，可能不对。

参观了传说中的天津，破是比我想象的稍破那么一点，不过比邯郸强。
参加了一次博客大赛就被论坛收编，加入了大学里的社团，挺不错的。

我真正得知道了微积分是那么有用，以前不敢想象能求解的问题用书上的公式竟然能搞定，咱也是学过微积分的人了，虽然很难。

认识了些朋友。

对微软的办公软件有深入的了解，因为要考试。

　失

没谈恋爱，很多人都开始了，目前还不知道有没有人结束。

体育颠球没及格，净上网啦，没怎么搞体育。

开学时想每天跑步，但只跑了两天，不是约数，就是2天。

学习上没用功，微积分勉强及格，普化玄点儿，英语就没学，光听听课。

可笑，可悲啊，大学的开端就这么完了，能怨谁呢？

过年回来一定努力。

50页纸111700元钱 一个女生四年大学的流水帐

发表于 2007-01-25
| 字数统计: 1.3k

　4年11万？当这两个数字从父母嘴里蹦出来时，静子（化名）的第一反应是“开玩笑”。她一直不相信这是自己大学4年的花费，直到半个月前收到母亲寄来的账本复印件。

　　那是本会计专用的账册，深蓝皮儿、A4纸大小、红绿色表格，密密麻麻记了50页，附带着工商银行的汇款单。共计：111700元。

　　第一笔开支是包括学费、军训费和保险费在内的7210元。那是5年前她离开青海西宁到重庆读大学时的报到费。向来是乖乖女的静子，除了学习很少关心其他事，因此花钱没什么计划，只知道“没了问爸妈要”。

　　结果不到一个月，原本两个月的1200元生活费花得精光。仅军训结束当天，她就逛街“血拼”了三四百块，理由是“憋得太久了，看到什么都想买”。

　　接到女儿的求救电话，当会计的母亲想出记账的方法，希望借此“教导女儿明白父母的不容易和勤俭节约”。作为普通公务员，她和丈夫年收入不足3万元。自从独生女上了大学，夫妇俩更是“花一分钱都精打细算”，4年中新衣服都很少添，三分之二的钱都花在了孩子身上。

　　可惜静子并未领会父母的用心。这个账本，她也就在大一寒假回家时翻了翻，印象里无非是“生活费500元、书本费300元”等小额数字。和往常一样，她花钱“有些凶”：MP3坏了，再买一个；三星手机破了，就换个西门子的。

　　尽管如此，母亲记账越发详细，从50元的手机充值费到150元的夹克衫钱，再到200元的火车票，都用黑色签字笔或者蓝色圆珠笔记得清清楚楚。

　　账本里的花销日趋增加。“难道是我小零小碎花得太多了？”静子事后一算，意外地发现就业一项竟占到了五分之一，比起4年间的零花也不逞多让。其中包括6种资格证书的报名费和资料费，英语、计算机、驾照……还有和专业毫不相干的导游证。

　　“多个证就多条路呗！”静子无奈地说，“因为文科的就业面太窄了。”据她介绍，同学中更夸张的还有法语等级证、美容师资格证等，“反正能考的大家都去考，也不管有没有用”。

　　进入大四之后，账本里的消费更是“触目惊心”，常常是千元一笔的支出。由于父母认为女孩子应该找份稳定的工作，静子开始奔波于各地的公务员考试。短短3个月间，她跑了广东、浙江和江苏等5个省份，每趟2000元。这还不包括前期的资料费、辅导费等。

　　虽然这笔钱算是白花了，因为静子并没有考上，但她认为是难以避免的。当时，几乎所有人都考疯了，即便是宿舍里家境最不好的同学，也花费数千元到各地找工作。

　　“怎么能说我是不孝之女呢？”静子委屈地辩解。一周前，账本的故事经《重庆时报》报道，被西南政法大学一位教育专家定性为“奢侈消费”，因为普通大学生的培养成本只有6万元。

　　对于这一数字，静子表示怀疑，认为偏低了。同宿舍12个人中，她自认只属于中等偏上水平。她承认自己有些大手大脚，爱买个手机链、头饰、包包等小东西，也常请朋友吃饭，但还算不上“败家”。

　　“很多钱是不得不花的，尤其在就业压力这么大的今天，高等教育的消费会越来越高。”她说。对于这一点，父母坚决支持。他们目前已拒绝了所有媒体的采访，防止给女儿带来更多的误解和压力。

　　静子公布账本的初衷，是希望学弟学妹们“节约花钱、体谅父母”。今年24岁的她难以想象父辈花几百元上大学的历史，“听说那时吃饭和住宿都是国家包，每个月还发生活费，现在怎么可能？”

　　她深知11万元对于工薪阶层父母的巨大压力。毕业前一天，母亲刚把当月的工资全部给她，几天后就又接到她的告穷电话。一时拿不出，母亲便跑去向朋友借。20分钟后，在电话里气喘吁吁地说：“孩子，给你寄了800块，先用着啊！”

　　“竟有人说我妈妈活该！”静子很是气愤，“谁家养孩子不花钱呢？”她听说，如今养个婴儿，连喂奶粉每月都要花2000元，而刚刚工作的她一个月也就拿这个数。

　　收到账本那一天，她开始吓了一跳。当脑海中浮现出父母日益苍老的模样，她禁不住哭了：“对不起，我真的不知道你们为我付出那么多。”

试用Google Docs & Spreadsheets

发表于 2007-01-20 | 分类于 网络爬虫
| 字数统计: 715

　　Google Docs & Spreadsheets是Google推出的一项在线使用的文字和表格编辑工具，相当于微软的Word和Excel，我使用这两个工具已经有一段时间了，谈谈自己的体会。 使用Google Docs & Spreadsheets最大的好处就是文档的安全有保障。我现在还没有自己的电脑，经常去网吧，做文字编辑用记事本或微软的Word，记录日常的花销用Excel然后保存到网盘里。我很少在浏览器里写东西，因为微软的IE不是很稳定，开几个窗口，只要有一个窗口出问题了，连带所有的窗口都要被强制关闭，里面的所有内容将不复存在，而每次都下载FireFox我又嫌麻烦。但即使这样也出现过让我痛心不已的情况，有一次我花了半个多小时写了些东西，突然网吧的线路出故障了，也就是说我写的东西无法保存到互联网上了，我又没有U盘，怎么办，只能气愤而无奈得下机走人了。自从用了Google Docs&Spreadsheets，我就不用担心类似的情况再次发生了，Google每隔几秒就会自动保存一次正在编辑的内容到Google服务器，如果出现网络连接故障，就会提示我之后编辑的内容可能无法保存到Google，我也就会停止编辑了，就算是有损失也不过是在几秒内编辑的内容会丢失，之前的内容都被稳妥得保存在Google服务器上了。以后买了电脑也可以避免因为学校停电和电脑死机而导致文档丢失。 除了文档的安全方面要比在电脑上使用编辑软件好以外，Google Docs & Spreadsheets还有其他好处。 一是可以实现文档的共享，可以选择一部分人来查看或编辑这些文档，这就避免了使用U盘，还能节省人力。 二是Google Docs & Spreadsheets提供了无限的文档存储空间，可以节省电脑的硬盘空间。 三是，当文档多了以后，可以不用分类，把工作、个人等等等类别的文档一股脑地交给Google，需要的时候，对Google说一两个关键词，让Google帮你找出来。 四是，Google Docs & Spreadsheets秉承了Google的一贯风格——简单、实用。微软的Excel确实功能强大，可以做图标，分类汇总，修饰表格等等，但对于普通用户来说只有少数的功能是常用的，其他的功能因一年也用不了几次。Google则仅将用户使用频率较高的功能加到自己的产品中，方便用户。 喜欢Google Docs & Spreadsheets。

我的忠诚度

发表于 2007-01-20 | 分类于 网络爬虫
| 字数统计: 702

　　我这个人似乎总是喜新厌旧，早年把Email在各个服务商间频繁换，先后用过Yahoo,Hotmail,mail.china.com,sohu,163,eyou甚至非主流的hushmail也凑过热闹，后来用上了Gmail，Email总算是固定下来了，博客却又开始换了走穴似的，blogger,sina,msn spaces,my opera,donews,blogger又换回这里了，可见我是一个忠诚度不高的用户，网络公司们可能不喜欢像我这样用户，但总有些公司的服务将我牢牢抓住，我很难丢弃他们的服务，尤其是当我用这些服务很长时间之后。

　　这些公司至所以吸引我都是因为他们已经成为我的大部分网络活动的平台，如果离开他们，我的网上生活就会变得很枯燥。

　　Gmail和Blogger就不用说了，这两个东西耗费了我的大量时间。我只是想说，我想要放弃图片存储和分享网站Yupoo和书签收藏网站365Key，回到曾经使用过的Flickr和del.icio.us。

　　想当初，我从flickr走向yupoo只是因为前者网速较慢，不仅是flickr.com的网页慢，而且放在flickr上的图片在博客中显示时也是比较慢的，我认为这是flickr的一大软肋，但最近我总是很怀念以前的flickr，怀念那种说不清，道不明的亲切感，怀念和世界上那么多人分享自己的图片，喜欢那么多的高质量的图片。Yupoo上的图片很少有专业人士发布的好图片，以生活照居多，而且当我需要将一张大图片缩小为较小的图片时，我发现Yupoo的压缩质量不能令人满意，我总是先在flickr压缩后再传到yupoo上。综合以上两点，于是乎我决定回到flickr，尽管他的速度较慢，没关系，还没到像Live Spaces那样让人无法忍受的蜗牛速度。

　　还是因为网速的问题，我在开始用书签记录上网经历的时候选择了365key，但后来觉得页面不是很友好，找个【登录】和【添加】按钮还得找半天，下载的【365KeyIt】也经常出错。我曾经无数次得看到网页上有del.icio.us的链接，我尝试了一下，比365好用多了，页面干净，服务稳定，互动性强，于是决定放弃用了很久的365，虽然有些可惜，因为每次在365加书签都是很费事的，我已经有那么多的书签了，可惜啊。

　　我可能是个完美主义者，总是想要更好的，于是也就经常放弃一些东西。我可能是个忠诚度不高的网民。

　Google提供的搜索记录服务，这是很不错的除了能告诉Google我更关心什么，以便在给出我搜索结果时考虑我的喜好，主要的还是能帮我保存搜索时间、次数和关键词，这些搜索记录似乎用处并不大，但至少我今天用这些数据算出了我平均每天用Google搜索多少次。

　　我是从2005年9月17日开始让Google记录我的搜索的，时至今日已经有16个月了，我今天第一次查看了Google记录的总搜索数――1592次。于是我算出了两个数字：

　　1 592 / 16 =99.5 　我平均每月用Google搜索近100次
　　1 592 / (16 * 30) = 3.31666667 　我平均每天用Google搜索3.2次

　　我是不是过于依赖Google了?
