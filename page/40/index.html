如何选择适合自己的杀毒软件

发表于 2008-08-01 | 分类于 小品软件 |
| 字数统计: 1.7k字 | 阅读时长 ≈ 5分钟

电脑安全问题困扰着几乎每一位电脑用户，个人信息、公司机密等都是黑客们的目标，把别人的电脑上的文件图标都换成熊猫会极大地满足某些人变态的优越感，所以用电脑，杀毒软件是比不可少的，尤其是在我国相关法律尚不健全的当下，防人之心不可无啊！ 有些人可能认为自己掌握了系统备份与还原操作，就算中了病毒，只要还原一下系统就OK，于是就裸奔，不装杀软和防火墙，但网络和移动存储设备一个不少，如果再加上不知Firefox为何物，如果再再加上经常上一些不良网站，那这就几乎等同于“新形势下的自杀”。因为，在互联网早期，病毒的主要目的是破坏电脑的正常工作，以实现造毒者变态的优越感，因为那时人和电脑结合得还不是那样紧密，既没有网上信用卡，也没有网游账户，但随着这些东西的出现，它们正在成为黑客盗取的对象，企业的文件被盗造成的损失可能更大，而这种新的病毒行为是在用户不知不觉中发生的，用户感觉不到什么资料就被盗走了。再加之，有些病毒甚至会破坏电脑的硬件设备，所以裸奔是绝对不可取的。 如何为自己的电脑选择一款杀毒软件？ 现在大多数的杀毒软件已经不仅仅能查杀病毒了，而是集防病毒、防木马、网络防火墙等功能于一身，这些功能并不是对所有人都适用的，比如对于使用基于网页的电子邮箱的人来说防垃圾邮件的功能就是鸡肋，对于办公电脑来说家长控制功能纯属多余，所以每个人都应该选择适合自己的安防组合。图省心的就用安全套装，一款软件提供多种保护，愿意自己搭配的就自己组合，比如我就是，防病毒防木马用Avira(红伞)，就图红伞的高查杀率，网络防火墙用风云，图他的安静和不错的防ARP攻击能力。有些杀软为了商业目的，自己炒作一些名词，比如QQ号保护，优盘保护等等，那都是扯淡，忽悠人的玩意儿，我不主张因为这些名词而左右杀软的选择，其实只要电脑具有防病毒，防木马，网络防火墙（使用Windows自带的防火墙也可以，但局域网用户推荐安装专业的第三方防火墙，因为自带防火墙不能防ARP攻击，当然如果是小型局域网可以绑定mac，像校园网这样的网还是装一个墙吧）我觉得就已经足够了，没必要太折腾自己和机子。提一下的是HIPS这个东西，它是较晚出现的安全类软件，能监控系统中的软件的一些动作，然后问用户是否允许，这种东西我认为只是一些泡安全论坛的人的玩具，并不适合普通用户，因为提问太多，无相关知识的用户贸然选择是或者否可能会造成比中病毒更麻烦的情况，干嘛如此的降低自己的使用体验。 商用杀软和非商用杀软。商用杀软的最终目的是给予终端员工最长的工作时间，确保文件的安全。就是说，无论电脑遭到何种攻击，都要尽可能不中断用户的正常使用，需要什么操作的话，下班后留给企业IT管理员去处理，要在上班时间内替老板从员工身上榨取最长的劳动时间；再有就是文件的安全，企业用户更担心自己的文件被删被盗，所以像赛门铁克这样的企业级杀软的默认操作就是隔离，而非删除，这给了用户回复被感染文件的可能。同时商用杀软的普遍特征是安静，不到万不得已不会和用户对话，很少要求用户参与，也就是最少的打扰员工的工作，不会一会蹦出个小狮子打瞌睡，一会跳出一只癞蛤蟆对你眨眼；并且在一些查病毒杀率测试中此类杀软的查杀率都不是很高，这就是为了在不影响系统安全的前提下，不让杀软有过高的敏感度，以减少对用户的干扰。商用电脑的配置一般较高，所以一些专门针对企业级市场的杀软占用系统资源是较高的，比如我在我的学生机上安装了赛门铁克的端点防护后，就明显感到电脑跑不动这个大家伙，所以我又把它卸载了。 非商用杀软则是另一番景象了，为了给用户提供最大的安全保护，一般他们都会把病毒特征库搞得很大，查杀率很高，后果就是杀软非常敏感，经常要求用户参与进来，发现威胁时的默认操作通常是删除，为了吸引用户眼球，把界面做的很华丽，此类杀软一般占资源较少，安装文件也较小，能给低配置用户以流畅的使用体验和较高的心理安慰。 这两种分类只是大致如此分，并不能涵盖所有的情况。这两种软件有着不同的设计理念，建议不同用户根据自己的需要进行选择。但不建议非企业用户使用网络集中式商用杀软，因为这种软件一般需要管理员进行统一的设置才能发挥最大的威力。 如果是企业用户和有着类似要求的非企业用户，我推荐赛门铁克、麦克菲和趋势科技的软件，这些公司的企业级软件能够为企业提供很好的保护。前提是企业要有一支好的管理员团队和充分的软件预算，个人用户要有足够的能力设置软件并找到免费的升级许可（个人一般是支付不起此类软件的费用的）。 如果是一般的家庭用户，我推荐江民、金山、卡巴斯基、NOD32、F-secure。 如果是超在意电脑安全的狂人，我推荐Avira(红伞)、 Avast!、AVK。

创建一个“文本文档”.将下面这段测试代码复制到“文本”里，保存。 X5O!P%@AP[4\PZX54(P^)7CC)7}$EICAR-STANDARD-ANTIVIRUS-TEST-FILE!$H+H* _（网上的这段代码有很多版本，有些是误传，我提供的代码源自http://www.eicar.org/anti_virus_test_file.htm是欧洲计算机防病毒协会官方网站上的权威版本）_ 测试等级: 特等：复制完代码后便提示内存有病毒 优等：刚保存完就提示病毒(或者直接删除) 中等：保存后几秒提示病毒(或者直接删除) 下等：需自己启动病毒扫描查杀才提示病毒(或者直接删除) 劣等：无论怎么扫描都无法提示病毒(或者直接删除) 以上代码和测试等级仅供参考，但我个人建议企业用户及在意信息安全的个人用户：如果测试结果是下等或劣等，请安装或更换你的防病毒软件。 我的电脑上安装了赛门铁克公司的某款软件，在点击保存瞬间就将文本文件自动删除,按照上述标准属于优等。
