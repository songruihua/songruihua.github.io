和谐 -> 水产 党 -> 派对 俯卧撑 -> 运动 SB、妈、母、娘、逼 -> 文明用语 粪、屎 -> 代谢产物 放屁 -> 文明用语 社会主义 -> 初级阶段 人木又 -> 好五倍的权利 政府 -> gov 无界 -> 没边 GFW -> Wall 煤矿 -> 黑色燃料基地 坦克 -> 打炮车 独裁 -> 一把剪刀 领导人 -> 擎天柱 垃圾、lj -> 废品 公安、警察-> 平平安安 藏 -> hide 网监 -> 太监 人大 -> 议会

摘自维基百科

以第一个关键词“和谐”为例，如果一位读者的发表了这样的评论”我们一定要共建和谐社会”，那么当评论发表之后，网页上显示的将是“我们一定要共建水产社会”，这就是关键词过滤。 我想你可能知道该网站为什么要过滤关键词，我就不做赘述了，但你却不一定知道： 1.为什么列表中的词组需要被过滤 2.为什么要这样“翻译” 下面我仅凭个人的肤浅的理解能力揣测一下该网站管理员的心里所想： 和谐 -> 水产： 过滤原因：莫谈国是（继“3代表”之后的又一个流行的政治词语） 翻译含义：和谐–>河蟹–>一种“水产” 党 -> 派对： 过滤原因：莫谈国是 翻译含义：“政党”的英文是party 俯卧撑 -> 运动：增强臂力的一种辅助性体育运动。 过滤原因：6月28日，贵州瓮安县出现严重打砸抢烧突发性事件。关于事情起因和经过，网上一度有各种传言。7月1日晚，贵州召开新闻发布会，新华网做了在线直播。发布会上贵州省公安厅发言人在介绍调查情况时说到，在李树芬溺水之前，与其同玩的刘某曾制止过其跳河行为，见李心情平静下来，刘“便开始在桥上 做俯卧撑，当刘做到第三个俯卧撑的时候，听到李树芬大声说‘我走了’，便跳下河中”。 几分钟后，8时整，天涯杂谈就出现了帖子：《我是来做俯卧撑的》。 由此，“做俯卧撑”已不仅是体育运动，而被网友恶搞，连夜火起来，用来表达“对某事不便或不愿发表意见”之意，例句：不要再吵了，大家做做俯卧撑不好吗？ 翻译含义：还原其本意 社会主义 -> 初级阶段： 过滤原因：莫谈国是 翻译含义：我国正处在并将长期处于社会主义初级阶段 人木又 -> 好五倍的权利 ： 过滤原因：略 翻译含义：某外交官有次跟美国人斗口，说出来了“中国人口总数是美国的五倍，所以中国的人木又状况乘以五，就是美国的五倍。所以，中国的人木又状况就比美国好五倍。” 政府 -> gov ： 过滤原因：莫谈国是 翻译含义：”政府”的英文是government，通常其简写是gov 无界 -> 没边 ： 过滤原因：据传是一个软件的名字，由国外反华势力（据传是法车仑工力组织，或有米国政府提供支持）制作，专门用于在大陆地区“不受政府监控”地使用国际互联网。 翻译含义：略 GFW -> Wall ： 过滤原因：因特网上有很多有害中国人民心理健康的内容，为了保护大陆网民，有人在某处设立了这么一个东西，用于过滤有害内容进入大陆网络，其全称是Great Fire Wall。“长城”的英语是Great Wall，计算机网络安全软件是Firewall（防火墙），GFW巧妙地将二者组合在一起，俗称“防火长城”。 翻译含义：简称 煤矿 -> 黑色燃料基地： 过滤原因： 不想网民联想到被活埋在地下的矿工 翻译含义：略 坦克 -> 打炮车： 过滤原因：略，如确想知道请问一些70后或年纪更大的人 翻译含义：略 独裁 -> 一把剪刀： 过滤原因：不许说邻国坏话 翻译含义: 一个人拿着一把剪刀去裁剪衣服，谓之“独裁”。 领导人 -> 擎天柱 ： 过滤原因：不许网民拿总统开涮 翻译含义：擎天柱是上世纪八十年代流行的动画片《变形金刚》中的一个机器人，是反派汽车人集团的领导人。 公安、警察-> 平平安安： 过滤原因：莫谈国是 翻译含义：警察保一方平安 藏 -> hide： 过滤原因：西藏是我国领土不可分割的一部分，打到达赖！ 翻译含义：“隐藏”的英语是hide 网监 -> 太监： 过滤原因：莫谈国是（网监是公安部门的网络监察，负责侦查犯罪活动。） 翻译含义：略 人大 -> 议会： 过滤原因：莫谈国是 翻译含义：中国的人大跟西方的议会的职责差不多 SB、妈、母、娘、逼、粪、屎 、放屁、垃圾、lj 等 过滤原因：有很多网民在评论时使用这些粗俗的词语 翻译含义：略

流氓网络电视PPS

发表于 2008-12-20 | 分类于 小品软件
| 字数统计: 503

当两台电脑都开着网络电视的时候，我的电脑就会在网上蜗行了。不得已也装了网络电视，先后下载了PPLive和PPStream两款网络电视。 记得以前也装过此类软件，觉得广告没那么多，这次装后，发现在选中节目后会先放一段广告，时间大约有30秒，要说不加广告企业无法生存，这也无可厚非，但是企业以牺牲用户体验为代价就得不偿失了，新浪、搜狐等门户以前在首页上弄弹窗广告、浮动广告，现在更是发明了什么背投广告、悬浮广告，严重干扰访问者视线、腾讯强制弹出新闻，并且不允许非会员用户关闭这个东西，明知会干扰用户，还要藉此来向用户要钱。印象中，优酷的广告不错，点开一个视频，只会在用户暂停的时候和视频旁边显示广告，不会在视频播放之前出广告。 互联网企业的这种行为，导致网上出现了很多“去广告版”和补丁，搜索谁家的软件去广告版最多，说明谁家的软件最不顾用户体验，现在我更是发现了一个我认为最“流氓”的广告模式，来自PPS。 无论什么广告，都是在用户访问该网页或打开某软件后才会出来，但唯独有一家企业——PPS的老板，把个PPS搞得无比恶心，只要是安装了PPS，即使不开着看片，也会蹦出广告，原来以为是PPS强加的一个广告进程，后来发现原来是一个屏幕保护程序，在安装时默认是安装的，同时默认的还有随windows自动启动。真是无所不用其极。唯一不及腾讯流氓的是，腾讯把QQ放在了开始菜单里Windows Update旁边，PPS没有。
